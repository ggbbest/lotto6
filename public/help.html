2. 로또 참여하기

game 접속 후 CONNECT 버튼을 클릭하면 메타마스크의 주소 정보를 가저옵니다.
메타마스크의 주소 정보를 가저온 화면 입니다. 숫자 6개를 선택하고 베팅코인수를 선택 한 후 추첨 버튼을 클릭합니다.
메타마스크에서 구매 금액을 확인 한 후 확인 버튼을 클릭 합니다.
화면에 숫자가 잠깐 보이고 좌측과 같은 화면이 나옵니다. 게임에 성공 하셨습니다.
위 화면에서 KLAY인경우는 트랜젝션ID 의 파랑색 글씨를 클릭하시면 scope 로 이동합니다. 하단 UTF-8 항목 확인 하시면 입력한 번호가 나옵니다. 3 13 23 33 43 24
(위 화면과 아래 화면은 다른 번호를 선택한 (C4EI, KLAY)것 입니다.)

(C4EI 경우는 익스플로러 작업 중입니다. )

구매하신 번호 리스트


로또 참여하기에서 CONNECT 버튼을 클릭하여 메타마스크의 주소 정보를 가저온 경우 Account: 우측 주소를 클릭하시면 구매하신 번호 리스트를 조회 가능 하십니다. 최대 30일 까지 표시 됩니다.
